IMPORTANT: the python dependencies for executing this html script are:

			Library:	Command:
				
			flask 		pip install flask
			pandas 		pip install pandas
			numpy		pip install numpy
			PIL		pip install pillow
			joblib		pip install joblib
			math		pip install math
			easygui		pip install easygui
			tensorflow	pip install tensorflow

To run this html script: 
		1) change the current working directory to C:/.../html_app.
		2) enter the command 'flask run'
		3) enter the provided address into your browser bar. EX: http://127.0.0.1:5000
		4) enter the latitude/longitude coordinates 
			a) integer values in [-90,90] and [-180,180] respectively
		5) click 'Submit Query'
		6) upload an image with the prompt
